CSS 3 - Feuille de style en cascade

Il existe trois moyens de faire du style à l'aide du CSS :

  • 1/ Le style en ligne (directement dans la balise)
  • 2/ Le style dans l'en-tête (dans la balise <head>)
  • 3/ Le style dans une feuille de style externe.
